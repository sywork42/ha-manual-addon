# Appliance Manual Add-on

Home Assistant add-on that lets you scan a QR code on an appliance and get an interactive manual with videos, PDFs, and troubleshooting — with in-page file uploads from your phone.

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**
2. Click the three-dot menu (top-right) → **Repositories**
3. Add this repository URL: `https://github.com/YOUR_GITHUB_USERNAME/ha-manual-addon`
4. Close the dialog. The **Manual Uploader** add-on will appear in the list.
5. Click it → **Install** → **Start**
6. Open the add-on's web UI to start creating appliance manuals

See [manual-uploader/README.md](manual-uploader/README.md) for full usage.
